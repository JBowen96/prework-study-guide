CSS

  • There are a lot of options available when making visuals in CSS.
  • Each element of the HTML can have specific visuals defined in CSS.
  • You can add classifications, such as "card" which can be appplied to multiple items.
  • A margin is on the outside, a padding is on the inside.

Git

  • You must use "git" at the beginning of every Git command.
  • Here are some commonly used git commands:
  • git checkout (main as example)
  • ^ this checks out "main"
  • git pull origin (main as example)
  • ^ this pulls the latest branch of "main"
  • git checkout -b feature/"name here"
  • ^ this creates a new feature of "name"
  • git status
  • ^ shows the current branch you are on
